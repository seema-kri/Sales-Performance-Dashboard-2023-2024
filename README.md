

# 📊 Sales Performance Dashboard (2023 vs 2024)  
### Excel-Based KPI & YoY Analysis Dashboard

🚀 Built an interactive Excel dashboard to analyze sales performance, track KPIs, and uncover year-over-year growth trends for business decision-making.

---

## 📊 Key Highlights

- 📈 Achieved **+30% Sales growth** and **+34% Profit increase** (2024 vs 2023)  
- 📦 Identified **+24% increase in quantity sold** with controlled **+11% cost growth**  
- 💡 Revealed improved **profit margins through cost optimization**  

---

## 🏗️ Dashboard Capabilities

- KPI tracking: **Sales, Profit, Quantity, Cost**  
- 📊 Year-over-Year (YoY) comparison (2023 vs 2024)  
- 📈 Monthly trend analysis  
- 🌍 Region-wise and category-wise performance  
- 🎯 Interactive filtering using slicers  

---

## ⚙️ Tools & Techniques

- **Microsoft Excel** (Pivot Tables, Pivot Charts)  
- **Functions:** IF, TEXT, ABS, TODAY  
- **Calculated Fields & GETPIVOTDATA**  
- **Slicers & Dashboard Design**  

---

## 🔄 Approach

- Cleaned and structured raw data using Excel Tables  
- Built KPI metrics and YoY calculations using Pivot Tables  
- Designed dynamic indicators for growth/decline  
- Created an interactive dashboard for easy exploration
